These are the available settings for this mod.
(When a parameter is required, it must be written in the quickbox without leading nor trailing spaces) Mandatory parameters will appear <ike this> Optional parameters will appear [like that]
Enable - Enable this mod
Teleport <true|false> - Enable/disable teleport to partner.
T.Price <integer> - If teleport enabled, define its cost.
T.Time <seconds> - If teleport enabled, how many seconds it takes.

Disable - Disable this mod.
Wed Fee <integer> - How much does the ceremony cost.
Div Fee <integer> - How much the divorce cost
Cheat.P <true|false> - Engaged/married people will receive penalties if try to engage w/somebody else.

FormalW <true|false> - People must have formal wear to participate in the ceremony
SameSex <true|false> - Allow gay couples
