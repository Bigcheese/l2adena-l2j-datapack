These are the available settings for this mod.
(When a parameter is required, it must be written in the quickbox without leading nor trailing spaces) Mandatory parameters will appear <ike this> Optional parameters will appear [like that]
Enable - Enable this mod
Frequecy <integer> - Chances in % for a mob to become champion.
MinLev <integer> - Minimum level allowed for a mob to spawn as champion.
MaxLev <seconds> - Maximum level allowed for a mob to spawn as champion.

Disable - Disable this mod.
xHP <integer> - HP multiplier for champion mobs.
HPregen <float> - HP regeneration rate multiplier
xAtk <integer> - Attack multiplier for champions (affects matk and patk).

Reward <integer> - Chances in % for special champion drops. Unaffected by server multipliers.
ItemId <integer> - ItemId of the rewards to drop.
ItmQty <integer> - Maximum amount of special items to drop.
AtkSpd <integer> - Attack speed multiplier for champions (affects matkspd and patkspd).
